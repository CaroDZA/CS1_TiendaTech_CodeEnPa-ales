**Descripción:**
Sistema de punto de venta para tienda de tecnología con gestión de inventario, ventas, clientes y personal. Incluye sistema de fidelización, múltiples métodos de pago y control de roles.

**Requerimientos:**
Lenguaje: Java 8+
IDE: NetBeans

*Instrucciones de compilación y ejecución:*
Abrir proyecto en NetBeans
Presionar F6 o Run
Iniciar sesión con credenciales de prueba
CAJERO = **usuario:** mcajera **contraseña:** cajera123
SUPERVISOR = **usuario:** jsupervisor **contraseña:** super123

*Realizar una Venta:*
Seleccionar productos → Agregar al carrito
Ir a Carrito → Seleccionar cliente (opcional)
Procesar Pago → Elegir método de pago
Confirmar y generar factura

*Gestionar Inventario:*
Menú → Inventario
Agregar/Eliminar/Actualizar productos
Guardar cambios en CSV

*Gestionar Clientes:*
Menú → Clientes
Buscar, registrar o canjear puntos
